# Hunter Bot

Discord бот для сервера с системой приватных голосовых комнат, верификацией, XP, логированием и статистикой.

---

## Требования

- Python 3.10+
- [FFmpeg](https://ffmpeg.org/download.html) — для музыкального плеера (добавить в PATH)
- [Deno](https://deno.land) — для yt-dlp (обход YouTube защиты)

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

Создай `.env` в корне проекта:

```env
BOT_TOKEN=твой_токен_бота
```

## Запуск

```bash
python main.py
```

## Команды

### Для всех участников

| Команда | Описание |
|---|---|
| `/help` | Список всех доступных команд |
| `/rank [@участник]` | Уровень и XP участника |
| `/top` | Топ участников по XP |
| `/music [запрос]` | Воспроизвести трек в голосовом канале |

### Только для администраторов

| Команда | Описание |
|---|---|
| `/set_info [канал]` | Канал оповещений о входе/выходе |
| `/set_verify [канал] [гость] [участник]` | Настроить верификацию |
| `/set_new_channel [канал]` | Лобби-канал для приватных комнат |
| `/set_music_channel [канал]` | Канал для музыкального плеера |
| `/set_log [канал]` | Канал для логов сервера |
| `/set_admin_log [канал]` | Канал для логов админских действий |
| `/set_level_role [уровень] [роль]` | Роль за достижение уровня |
| `/give_xp [@участник] [кол-во]` | Выдать XP вручную |
| `/userinfo [@участник]` | Подробная информация об участнике |
| `/serverinfo` | Статистика сервера |
| `/send_message [@участник] [текст]` | Отправить ЛС от имени бота |

## Функции

### Верификация
При входе на сервер участник получает роль гостя и видит только канал верификации. После нажатия кнопки и ввода ника в формате `Ник (Имя)` — получает роль участника и доступ к серверу.

### Приватные голосовые комнаты
Зайди в лобби-канал — бот создаст комнату и отправит панель управления прямо в канал. Настройки сохраняются и применяются при следующем создании.

Панель управления:

| Кнопка | Действие |
|---|---|
| ✏️ Название | Переименовать (пресеты или своё) |
| 👥 Лимит | Установить лимит участников |
| 👁 Видимость | Скрыть/показать канал |
| 👑 Владелец | Передать права |
| 📹 Видео | Разрешить/запретить видео |
| 🎤 Voice | Разрешить/запретить голос |
| 🔨 Баны | Забанить участника в комнате |
| ⬆️ Поднять | Поднять панель вниз (кулдаун 3 мин) |

### XP система
- За каждое сообщение (кулдаун 60 сек): 10–25 XP
- За каждые 5 минут в голосовом канале: 10 XP
- При повышении уровня бот уведомляет в чате
- Можно привязать роль к уровню через `/set_level_role`

### Стрики войса
Каждый день захода в голосовой канал увеличивает стрик на 1. Стрик отображается в нике: `CodeWriter (Борис) 🔥5`. Обновляется в 00:00 по МСК.

### Логирование
Два отдельных канала:
- Лог сервера — сообщения, голосовые перемещения, изменения участников
- Админ-лог — баны, кики, роли, каналы

### Автобэкап
БД автоматически копируется каждые 24 часа в папку `backups/`. Хранятся последние 7 копий.
