# Hunter Bot

Discord бот с системой приватных голосовых комнат, верификацией, XP, логированием, статистикой и интеграцией Dota 2 / CS2.

---

## Требования

- Python 3.10+

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

Создай `.env` в корне проекта:

```env
BOT_TOKEN=твой_токен_бота
STEAM_API_KEY=твой_steam_api_ключ
FACEIT_API_KEY=твой_faceit_api_ключ
```

Steam API ключ: [steamcommunity.com/dev/apikey](https://steamcommunity.com/dev/apikey)  
Faceit API ключ: [developers.faceit.com](https://developers.faceit.com)

## Запуск

```bash
python main.py
```

## Функции

### Верификация
При входе участник получает роль гостя. После нажатия кнопки и ввода ника в формате `Ник (Имя)` — получает роль участника.

### Приватные голосовые комнаты
Зайди в лобби → бот создаёт комнату с панелью управления. Настройки сохраняются для следующего раза.

Панель: ✏️ Название · 👥 Лимит · 👁 Видимость · 👑 Владелец · 📹 Видео · 🎤 Voice · 🔨 Баны · ⬆️ Поднять

### Музыкальный плеер
`/music [запрос]` — ищет трек на YouTube через Lavalink. Прогресс-бар обновляется каждые 5 секунд.

### XP система
- Сообщения: 10–25 XP (кулдаун 60 сек)
- Войс: 10 XP каждые 5 минут
- Стрики: 🔥N в нике за ежедневный заход в войс

### Dota 2 / CS2
- Привязка Steam/Faceit аккаунтов
- Статистика, последние матчи
- Автоуведомление о начале матча с составом и статистикой всех игроков
- Уведомление об изменении ранга
- Недельная статистика гильдии

### Логирование
- Лог сервера: сообщения, голосовые перемещения, изменения участников
- Админ-лог: баны, кики, роли, каналы

### Автобэкап
БД копируется каждые 24 часа в `backups/`. Хранятся последние 7 копий.
